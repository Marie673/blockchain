# blockchain
# Requirement
 Vagrant
 
 Virtual Box

# Usage
Vagrant, VirtualBoxがインストールされている環境であれば，Vagrantfileがある場所にて

`vagrant up`
で仮想環境が新しく建てられます．

新しい環境がたったら，
`vagrant ssh`
簡単に仮想環境にssh接続できます．

仮想環境を停止する際は
`vagrant halt`
で停止できます

仮想環境を壊したいときは
`vagrant destroy`
で壊せます
